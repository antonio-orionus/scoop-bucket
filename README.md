# Arroxy Scoop Bucket

Install Arroxy on Windows via [Scoop](https://scoop.sh):

```
scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket
scoop install arroxy
```

Manifests in this bucket are auto-bumped on each [Arroxy](https://github.com/antonio-orionus/Arroxy) release by GitHub Actions.
